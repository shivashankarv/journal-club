# 🎮 Paper Mastery Game Skill

Your gamified coach for testing, building, and mastering.

## Purpose

Transform learning into engaging challenges:
- **Quiz**: Test your understanding
- **Build**: Implement in sandbox
- **Challenge**: Solve hard problems
- **Capstone**: Build complete model

## When This Skill Activates

When you want to test or build:
- "Test me" or "PLAY [1]"
- "Let me implement this"
- "Unlock challenges"
- "Time for capstone?"

## 4 Learning Phases

1. **Concept Quizzes** - Test understanding
2. **Sandbox Implementation** - Build in real Python
3. **Challenges** - Optimization, edge cases
4. **Capstone** - Full model from scratch

## Gamification Features

- **XP Points**: Earn for correct answers + implementations
- **Levels**: Novice → Apprentice → Journeyman → Expert → Master
- **Badges**: Achievements (No Bugs, Edge Case Hunter, etc.)
- **Progress Tracking**: See your advancement
- **Streaks**: Consecutive correct answers unlock bonuses

## How to Use

### Test Understanding
```
"PLAY [1]" (or "Test me")
→ Quiz on concept 1
→ Immediate feedback
→ XP earned if correct
```

### Implement
```
"START SANDBOX"
→ Coding challenge in real Python
→ Unit tests provided
→ Real feedback (pass/fail)
→ XP earned for passing
```

### Get Harder
```
"HARD MODE" or "UNLOCK CHALLENGES"
→ Edge case hunting
→ Optimization problems
→ Research extensions
→ +2x XP available
```

### Final Project
```
"CAPSTONE"
→ Build complete model from scratch
→ Comprehensive test suite
→ Achieve MASTER status
```

```
Learn attention mechanism (Tutor)

```

## Key Features

- **Real sandbox**: Actual PyTorch/NumPy environment
- **Instant feedback**: Tests run immediately
- **XP-driven**: Motivation through progression
- **Adaptive difficulty**: System adjusts to your skill
- **Comprehensive testing**: Shape checks, gradient flow, edge cases
