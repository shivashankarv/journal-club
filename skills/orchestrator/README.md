# 🎭 Orchestrator Skill

The intelligent conductor of the Paper Mastery System.

## Purpose

Routes between other skills intelligently based on:
- Your learning style (Theorist, Practitioner, Balanced)
- Paper complexity
- Your progress and understanding
- Optimal learning path

## When This Skill Activates

Automatically whenever you interact with the system.

## What It Does

1. **Detects your learning style** on first interaction
2. **Analyzes the paper** you're learning
3. **Recommends optimal path** (which skill, what sequence)
4. **Routes to other skills** seamlessly
5. **Tracks progress** as you learn
6. **Adapts pacing** if you're moving too fast/slow
7. **Coordinates transitions** between skills

## How to Use

You don't need to explicitly use this skill - it works in the background. But understand:

- **You say**: "I want to master this paper"
- **Orchestrator does**: Analyzes and creates plan
- **Orchestrator says**: "Let me start with a Feynman summary"
- **Then activates**: Paper Tutor or Paper Mastery Game as needed

## Example Flow

```
Upload Paper
  ↓
[Orchestrator] Detect: Balanced learner, theory-heavy paper
  ↓
"Let's start with intuition"
[Activate: Paper Tutor]
  ↓
User: "Next section"
[Orchestrator monitors understanding]
  ↓
"Ready to test yourself?"
[Activate: Paper Mastery Game]
  ↓
[Orchestrator coordinates transitions smoothly]
```

## Key Features

- **Smart routing**: Decides when to teach, when to test
- **Adaptive pacing**: Speeds up or slows down based on your performance
- **Progress tracking**: Knows what you've learned
- **Seamless transitions**: You never think "which skill should I use?"
- **Personalization**: Adapts to your learning style

## Configuration

You can customize:
- Learning pace (fast/normal/slow)
- Learning style preference (Theorist/Practitioner/Balanced)
- Difficulty level (easy/normal/hard)
- Feedback style (concise/detailed)
