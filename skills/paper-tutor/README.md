# 🎓 Paper Tutor Skill

Your personal professor for deep, pedagogical learning.

## Purpose

Teach you to understand research papers deeply using:
- Feynman-style explanations (clear, intuitive)
- Mathematical derivations (step-by-step)
- Pseudocode implementations
- Domain extensions (connect to your research)

## When This Skill Activates

When you want to understand concepts deeply:
- "Explain this concept"
- "Next section"
- "Show me the math"
- "How do I apply this to peptides?"

## 7 Learning Phases

1. **Feynman Summary** - Intuitive overview
2. **Section-by-Section** - Deep explanations
3. **Mathematical Details** - Equations + derivations
4. **Architecture & Background** - How it's built
5. **GitHub Code** - Implementation walkthrough
6. **Results Analysis** - Evaluate findings
7. **Extension** - Apply to your domain

## How to Use

### Phase 1: Get Started
```
"Give me the Feynman summary"
→ Tutor provides intuitive overview
```

### Phase 2: Go Deep
```
"Next section" or "Explain [concept]"
→ Tutor explains with math, code, examples
```

### Phase 3: Extend
```
"How do I apply this to peptides?"
→ Tutor creates adaptation plan
```

## Key Features

- **Feynman clarity**: No jargon, simple explanations
- **Multiple representations**: Words → equations → code → analogies
- **Bottom-up learning**: Prerequisites before advancing
- **Patient pacing**: You control speed with "next section"
- **Domain adaptation**: Extension to your research

```
Why: Scaling prevents gradient explosion
```
