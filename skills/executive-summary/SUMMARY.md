## Limitations & Design Choices

### By Design (Not Bugs)
- **Quizzes don't require perfect answers** → Concept testing, not memorization
- **No hints cost 0 first hint** → Encourages trying before peeking
- **Implementation challenges are hard** → Real learning is hard
- **Sandbox is simplified pseudo-code** → Not production code, but teaches concepts

### Known Constraints
- **Requires PDF upload** (can also take paper links + GitHub URLs)
- **Gamification works best for 5-15 concepts** (short papers)
- **Sandbox is CPU-bound** (no GPU, so toy data sizes)
- **Best with 5+ hours of dedicated study** (cramming = shallow learning)
