### Skill Commands

**Paper Tutor**:
```
"Next section"
"Explain [concept]"
"Show me the math"
"How do I apply this to [domain]?"
```

```
"PLAY [N]" (start concept N quiz)
"START SANDBOX" (begin implementation)
```

### Troubleshooting

**Stuck on concept?**
- Use Paper Tutor: "Explain this more intuitively"
- Ask for background: "What's the background for [concept]?"

**Failing tests?**
- Read the error carefully
- Use hints: "Give me a hint"
- Ask for pseudocode: "Show me the pseudocode"

**Bored?**
- Unlock hard mode: "HARD MODE"
- Jump to challenges: "Skip to challenges"
- Speed up: Tell system "I'm moving fast"

**Overwhelmed?**
- Slow down: "I need more time"
- More examples: "Give me more examples"
- Simpler explanations: "Explain more simply"
